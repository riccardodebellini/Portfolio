Ho creato questo sito per padroneggiare le basi, scrivendo ogni riga di codice a mano per comprendere a fondo le fondamenta su cui si regge il web moderno.

### Tech Stack
Un approccio "Back to Basics":
*   **HTML5 Semantico** – La struttura pura.
*   **CSS3 (Vanilla)** – Stile e layout senza pre-processori e [classi personalizzate](/progetti/material-components)
